# 🚀 FSM Project: Smart Field Operation

> **"Field work is a dot, Data is a line, and Insights are a plane."**  
> 모든 현장의 기록을 점으로 찍고, 이를 연결하여 비즈니스의 평면을 장악합니다.

본 저장소는 공간살균기 설치·AS 운영의 **엔지니어링 표준**을 정립하기 위한 프로젝트 허브입니다.  
단순한 기록을 넘어, 데이터의 무결성과 확장성을 최우선으로 하는 엔터프라이즈급 설계를 지향합니다.

---

## 🧠 Core Philosophy (우리의 철학)

1.  **WorkOrder First:** 모든 비즈니스는 WorkOrder의 상태 변화를 통해 증명됩니다.
2.  **Snapshot over Reference:** 과거의 기록은 영원히 변치 않아야 합니다. (데이터의 무결성)
3.  **Strong Engineering:** 유연성이라는 명목하에 부채를 남기지 않습니다. (Anti-JSONB for Core)
