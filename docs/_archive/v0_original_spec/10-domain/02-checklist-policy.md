# 02 체크리스트 정책 및 표준 템플릿

## TL;DR
- **포맷:** PASS / FAIL / NA 3단계 선택 기반의 디지털 증빙
- **원칙:** 설치 전/후 환경, 성능, 고객 케어를 포함하는 5대 점검 원칙 준수
- **강제성:** 필수 항목 미입력 시 제출 불가, FAIL 선택 시 메모 입력 필수, 고객 서명 필수

---

## 1. 체크리스트 점검 5대 원칙 (Standard)

현장 품질의 표준화를 위해 모든 체크리스트 항목은 다음 기준을 바탕으로 설계합니다.

1. **설치 환경 점검:** 주변 영향 요소(누수, 과열, 공간 확보 등) 사전 확인
2. **설치 품질 확인:** 제품의 수평/수직, **평탄도**, 고정 상태 등 물리적 완성도 점검
3. **기능 정상 작동:** 전원, 소음, 기초 동작 등 즉각적인 성능 확인
4. **안정성 모니터링:** 10분 이상 가동 후 임계 수치(수위, 온도 등) 변화 기록
5. **고객 케어:** 주요 기능 설명 및 주의사항 대면 안내 완료 여부

---

## 2. 🛠️ 설치(INSTALL) 표준 템플릿

WorkOrder 생성 시 자동으로 생성되는 필수 점검 항목입니다.

| ID | 점검 항목 | 검증 기준 / 비고 | 필수 |
| :---: | :--- | :--- | :---: |
| S01 | 협의된 설치 위치 확인 | 고객 요청 위치와 실제 설치 위치 일치 여부 | ✅ |
| S02 | 설치 환경(누수/과열) 점검 | 주변 누수 위험이나 이상 발열 요소 유무 확인 | ✅ |
| S03 | 전원 공급 및 정격 전압 확인 | 콘센트 고정 및 전용 회로 확인 | ✅ |
| S04 | 제품 모델 및 시리얼 일치 | 배정된 모델명과 실물 시리얼 번호 매핑 | ✅ |
| S05 | **평탄도** 및 수평 정렬 | 바닥면 밀착 및 육안 기준 수평 정렬 완료 | ✅ |
| S06 | 고정 및 외관 상태 확인 | 브라켓 고정력 확인 및 파손/스크래치 유무 | ✅ |
| S07 | 초기 구동음 및 동작 확인 | 부팅 시 이상 소음 유무 및 기본 동작 테스트 | ✅ |
| S08 | **[성능] 10분 가동 후 수치** | 10분 가동 후 목표 수위/온도 도달 여부 확인 | ✅ |
| S09 | 에러 코드 발생 여부 | 시스템 자가 진단 시 에러 코드 노출 유무 | ✅ |
| S10 | 사용자 교육 및 안내 | 주요 기능 설명 및 긴급 차단법 대면 안내 | ✅ |
| S11 | 현장 복구 및 주변 정리 | 설치 쓰레기 회수 및 주변 정리 정돈 완료 | ✅ |
| S12 | **고객 서명 완료** | **최종 승인 서명 (고객 성명 포함)** | ✅ |

## 3. 🔧 AS(SERVICE) 표준 템플릿

수리 및 점검 방문 시 적용되는 표준 항목입니다.

| ID | 점검 항목 | 검증 기준 / 비고 | 필수 |
| :---: | :--- | :--- | :---: |
| A01 | 접수 증상 재확인 | 고객 상담 내역과 실제 증상 대조 확인 | ✅ |
| A02 | 대상 기기 시리얼 확인 | 유지보수 대상 기기의 식별 정보 일치 여부 | ✅ |
| A03 | 기본 인프라 연결 점검 | 전원/통신선 등 외부 연결 상태 우선 체크 | ✅ |
| A04 | 원인 특정 및 재현 테스트 | 이상 증상 재현 여부 확인 및 기록 | ✅ |
| A05 | 수리 및 소모품 교체 수행 | 부품 교체, 설정 변경 또는 클리닝 완료 | ✅ |
| A06 | 조치 결과 및 재생 성능 확인 | 증상 해소 여부 및 10분 가동 테스트 통과 | ✅ |
| A07 | 제품 외관 및 청결 상태 | 조치 과정에서 발생한 오염 및 파손 점검 | ✅ |
| A08 | 조치 설명 및 사후 관리 안내 | 처리 내용 및 향후 필터 교체 등 일정 안내 | ✅ |
| A09 | 후속 조치(방문) 필요성 | 추가 부품 수급이나 재방문 필요 여부 고지 | ✅ |
| A10 | **고객 서명 완료** | **최종 승인 서명 (고객 성명 포함)** | ✅ |

## 📸 4. 증빙 사진 촬영 가이드 (권장)
- **조치 전 사진:** 현장 도착 시 초기 상태 (1장 이상)
- **조치 후 사진:** 최종 설치 완료 및 정리된 상태 (1장 이상)
- ※ Admin 설정에 따라 특정 항목에 대한 사진 촬영이 강제될 수 있음 (Phase 1.5)

---

## ⚖️ 5. 운영 및 완료 정책
- **FAIL 처리:** 필수 항목이 FAIL이어도 사유 입력 시 제출 가능 (단, 관리자 검토 대상)
- **메모(Note) 강제:** FAIL 선택 시 조치 계획 또는 사유 입력 미완료 시 제출 버튼 비활성화
- **미래 확장성 (Gate 2):** 향후 제품 마스터 관리와 연계하여 **제품 카테고리별 특화 템플릿**이 자동 호출되도록 고도화 예정
