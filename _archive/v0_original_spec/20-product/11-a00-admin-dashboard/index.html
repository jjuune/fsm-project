    A-00 Admin 대시보드

    TL;DR

    • Admin 로그인 후 첫 랜딩 화면. 팀별 운영지표 MVP 카드 제공.
    • 성과평가 목적 아님. A-02 목록 필터 진입 허브 역할.

    📌 상세 스펙 위치

    A-00 Admin 대시보드의 화면 구성, 카드 지표, 인터랙션 상세 내용은 A-01 팀 관리 + A-00 Admin 대시보드 문서의 2. A-00 섹션을 참고하세요.

    📊 화면 요약

    항목 내용
    접근 역할 Admin (ORG 스코프)
    랜딩 위치 H-01 (My Home) → 역할이 ADMIN일 때 자동 이동
    핵심 컴포넌트 팀별 상태 카운트 카드, 지연 카운트, A-02 필터 링크

    연관 문서

    • A-01 팀 관리 + A-00 상세 스펙
    • Admin 작업 목록 (A-02)
    • 권한 매트릭스(RBAC)
