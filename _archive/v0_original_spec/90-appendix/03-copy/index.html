    용어 및 문구 세트 (UX Copy)

    TL;DR

    • 원칙: 짧고 명확한 문구, 부정적 단어(오류 등) 배제, 행동 유도형 버튼명 사용
    • 상태 표기(현장): UI는 OK / 이상 / 해당없음으로 표기 (저장값은 PASS / FAIL / NA 유지)

    👷 1. 기사용 문구 (M-02)

    상황 UI 문구 안내/토스트 메시지
    자동 저장 (없음) 저장 완료 ✅ (실패 시: 저장 실패 · 재시도)
    체크리스트 OK, 이상, 해당없음 이상 항목 메모가 필요해요
    고객 서명 서명 저장, 다시 받기 서명을 저장해야 제출이 가능합니다
    사진 업로드 사진 추가, 삭제 전/후 사진을 남겨주세요 (권장)
    완료 제출 완료 제출 완료 처리됐어요. 잠시 후 문서가 발송됩니다.

    🧑‍💼 2. 팀 관리자용 문구 (T-01~03)

    상황 UI 문구 토스트/메모
    KPI 카드 미배정, 진행중, 지연 기사 배정과 관리가 필요한 작업입니다.
    퀵 배정 기사 배정, 배정하기 기사가 배정되었습니다.
    재배정 기사 재배정, 변경 진행중인 작업은 재배정이 제한됩니다.
    필터/검색 현장/고객/주소 검색 검색 결과가 없습니다.

    👑 3. 본사 관리자용 문구 (A-02~04)

    상황 UI 문구 안내/상태 메시지
    팀 배정 팀 배정, 팀 변경 팀 변경 사유를 입력해주세요. (필수)
    취소 처리 취소 처리 작업이 취소되었습니다.
    PDF/문서 PDF 다운로드, 재발송 문서 생성 중입니다... / 준비됨 ✅
    발송 결과 발송 완료, 발송 실패 발송에 실패했어요. 재발송을 시도해주세요.

    📋 4. 공통 에러 및 안내 패턴

    • 네트워크 실패: 네트워크를 확인해주세요 · 재시도
    • 로그인 만료: 로그인이 만료됐어요. 다시 로그인해주세요.
    • 접근 제한: 이 작업은 볼 수 있는 권한이 없어요.
